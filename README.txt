This project should be executed in a Java IDE like Eclipse.
1. To create a file containing Token and Document Dictionary run the file "Tokenizer.java" in src folder.
2. The output is written to a text file "parsed_output.txt".
3. The text extracted from the given folder "ft911" is written to "TextValue.txt".
4. The document numbers are extracted to "docsText.txt" from the "ft911".

The output consists of tokens with unique ids assigned and document number with unique ids assigned.
